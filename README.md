# Mòdul Programació d'intel·ligència Artificial
## Tasca 3 pingu-palmer
Aquesta tasca consta d'un classificador sobre el conjunt de dades "seaborn" el qual conté diverses dades de 343 individus de pingüins de tres espècies diferents que es troben a tres illes de l'Arxipèlag Palmer de l'Antàrtida.
S'empren els següents models:
1. Regressió logística
2. Màquines de suport vectorial (SVM, Support Vector Machine)
3. Arbres de decisió (Decision Trees)
4. K veïnats més propers (KNN, k-Nearest Neighbours)
